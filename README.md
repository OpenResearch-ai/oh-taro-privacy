# 개인정보처리방침

**앱 이름:** 오! 타로 (Oh Taro)
**번들 ID:** ai.oo.taro
**시행일:** 2026년 3월 11일
**최종 수정일:** 2026년 3월 11일

## 1. 개요

오! 타로(이하 "본 앱")는 사용자의 개인정보를 소중히 여기며, 관련 법령을 준수하여 개인정보를 처리합니다. 본 개인정보처리방침은 본 앱이 수집하는 정보의 종류, 사용 목적, 보관 방법 및 사용자의 권리에 대해 설명합니다.

---

## 2. 수집하는 정보

### 2.1 자동으로 수집되는 정보

| 정보 유형 | 수집 목적 | 저장 위치 |
|-----------|-----------|-----------|
| 익명 UUID (기기 고유 식별자) | 사용자 세션 유지, 리딩 기록 연결 | Firestore (Google Firebase) |
| FCM 토큰 (Firebase 푸시 알림 토큰) | 푸시 알림 전송 (선택 동의 시) | Firestore (Google Firebase) |

### 2.2 사용자가 생성하는 정보

| 정보 유형 | 수집 목적 | 저장 위치 |
|-----------|-----------|-----------|
| AI 오라클과의 채팅 내용 | 타로 리딩 제공, 대화 맥락 유지 | Firestore (Google Firebase) |
| 타로 리딩 기록 (선택된 카드, 해석 내용) | 리딩 히스토리 제공 | Firestore (Google Firebase) |

### 2.3 수집하지 않는 정보

본 앱은 다음 정보를 **수집하지 않습니다**:

- 이름, 성별, 생년월일 등 인구통계 정보
- 이메일 주소
- 전화번호
- 주소 또는 위치 정보
- 결제 카드 정보 (결제는 Apple App Store / Google Play를 통해 처리됨)
- 사진, 카메라, 마이크 접근 데이터

> **참고:** 사용자가 채팅 대화 중 자발적으로 이름, 나이 등의 개인정보를 AI에게 공유하는 경우, 해당 정보는 대화 맥락 향상 목적으로 AI 처리에만 사용되며, 별도로 추출·저장·분석되지 않습니다.

## 3. 정보 사용 목적

수집된 정보는 다음 목적으로만 사용됩니다:

1. **서비스 제공:** 타로 리딩 세션 진행 및 개인화된 AI 응답 생성
2. **기록 보관:** 사용자가 이전 리딩 기록을 다시 확인할 수 있도록 저장
3. **서비스 개선:** 익명화된 사용 패턴 분석을 통한 앱 기능 개선
4. **알림 서비스:** 사용자가 동의한 경우 푸시 알림 전송
5. **기술 운영:** 오류 감지 및 서비스 안정성 유지

## 4. 제3자 제공 및 위탁

본 앱은 다음 제3자 서비스를 사용하며, 서비스 제공에 필요한 최소한의 정보만 전달됩니다.

### 4.1 Google Firebase (Alphabet Inc.)

- **목적:** 데이터 저장(Firestore), 푸시 알림(FCM), 앱 분석(Analytics)
- **데이터:** 익명 UUID, 채팅 내용, 리딩 기록, FCM 토큰
- **개인정보처리방침:** https://policies.google.com/privacy

### 4.2 Google Gemini API (Alphabet Inc.)

- **목적:** AI 기반 타로 리딩 텍스트 생성
- **데이터:** 사용자의 채팅 메시지 (API 호출 시 전달)
- **개인정보처리방침:** https://policies.google.com/privacy
- **참고:** Gemini API는 사용자 입력을 모델 학습에 사용하지 않습니다.

### 4.3 RevenueCat, Inc.

- **목적:** 인앱 결제 관리 및 구독 상태 추적
- **데이터:** 익명 사용자 ID, 구매 내역 (결제 정보 자체는 Apple/Google이 처리)
- **개인정보처리방침:** https://www.revenuecat.com/privacy

### 4.4 Apple App Store / Google Play Store

- **목적:** 인앱 결제 처리
- **데이터:** 결제 정보 (본 앱은 결제 정보에 직접 접근하지 않음)

**본 앱은 사용자 정보를 판매하거나, 위 서비스 제공자 이외의 제3자에게 제공하지 않습니다.**

## 6. 사용자의 권리

사용자는 다음 권리를 가집니다:

- **열람권:** 본인에 관해 수집된 정보 확인 요청
- **삭제권:** 수집된 모든 데이터 삭제 요청 ("잊혀질 권리")
- **처리 제한권:** 특정 목적의 데이터 처리 중지 요청
- **동의 철회권:** 푸시 알림 등 동의 기반 처리에 대한 동의 철회

**데이터 삭제 요청 방법:**
1. 앱 내 설정 화면에서 "데이터 삭제" 버튼 이용 (지원 시)

## 8. 보안

수집된 데이터는 Google Firebase의 보안 인프라를 통해 암호화되어 저장됩니다. 익명 UUID를 사용하여 실제 사용자 신원과 데이터를 분리합니다. 그러나 인터넷을 통한 데이터 전송은 100% 안전을 보장할 수 없으므로, 민감한 개인정보를 채팅에 입력하지 않도록 권장합니다.
